## Como executar
Pré-requisitos: npm / yarn
```bash
# Clonar o repositório
git clone https://github.com/cleversonmartins/podcastr-nlw5.git

# A primeira vez que for executar o projeto é necessário baixar as dependências
yarn

# Em um terminal rode o json-server
yarn server

# Com o json-server rodando
# Crie a build do projeto
yarn build

# Execute o projeto
yarn start

# Abra o navegador com o endereço abaixo
http://localhost:3000
```
